### 写 demo 的原因
说到轮播我想很多人都知道  
但是他(轮播)并不是一个简单的组件而已   
相反他同样能反应一个人的能力  
去年曾经用 Vue 的模式写过轮播,了解过了原理
但是 我觉得并不够  
因为  他有可以有很多纷繁复杂的要求,在这个 demo 中我会将其一一实现  
记于 2019.6.9


之前在 v 站也碰到过有人问 如何培养编程思维  
其实在写这 demo 的过程中 ,确实也可以培养一下,如何养成  
后续我将写一篇博客来引导一下
